Christmas Tree Speaker Star

This Chistmas Tree Speaker Topper is a sound reactive light bar that produces it's own bluetooth sound. Using a sparkfun sound sensor and neopixel LEDs, I was able to convert sound amplitudes into analog readings that produce different values at different frequencies. Using these frequencies and a clock, I was able to get the leds to change color based on the amplitude of the sound wave. The base is 3D Printed along with a Laser Cut Star, all programming was done in C++! Watch it in Action

Movie Connect Application

Movie Connect was created for movie lovers like myself. You start by taking a quick google survery which then stores your data into a parsable excel file. Based off of the data that is collected, you are then matched with users who like the same movies as you. The program uses a match_similarity algorithm that takes into account your genre tendencies, likeability towards specific titles, etc. View the Repository

Sound Reactive Neopixel Lightbar

Using an Adafruit sound sensor along with Neopixel LEDs, this light bar has several different modes. One being an insanely cool sound reactive mode that will light up the bar to a specific height based on the analog readings from the sound sensor! See it in Action
